# Tooling for cross-chain testing
EthereumZuri.ch 2023 

## Setup

- Install Woke:
    - `pip3 install -U woke`
- Install [Foundry](https://github.com/foundry-rs/foundry):
    - `curl -L https://foundry.paradigm.xyz | bash`
- Install node dependencies:
    - `npm ci`
- Generate pytypes:
    - `woke init pytypes`
